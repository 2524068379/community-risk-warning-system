**成果名称：**险“封”——基于VLM模型的社区风险预警智能体

**申报类别：**科技制作与技术应用类

**推荐院校：**江苏警官学院

**完成人：**徐雨轩、胡铭泽、余岳洋、王述哲、汪佳涵

**指导教师：**高谷刚、戚志如

**成果简介（500字以内）：**

社区风险多元，消防通道占用、电动车违规充电、异常聚集、可疑徘徊等发现滞后、联动不足，人工巡查”看不过来、看不明白、追不回去、难以追溯”。围绕这一痛点，本项目研制了”险封”社区风险预警智能体系统。视觉语言大模型（VLM）负责”看懂”与”说清”，可理解长视频、定位关键帧，并给出风险等级、证据片段与处置建议；Agent 负责”行动”，按预设流程自主完成感知、研判、复核与工单派发，遇到不确定场景还会调用规则知识库自我校正。二者分工协同，让系统像 24 小时在岗的”智能巡防员”，从”看得见”走向”看得懂、会处置”。工程上采用”前端轻量预筛 + 大模型深度研判”的分层结构：先用浏览器侧轻量目标检测与自适应截帧把”值得看”的画面筛出来，再交由 Qwen 多模态大模型与本地 VLM 做场景理解，形成”发现-研判-复核-处置”闭环。项目已落地软硬件原型，在警犬基地、物业小区及合作单位实测验证，可分钟级发现典型风险并分级告警。团队围绕核心算法、系统软件和应用方案，同步布局论文、专利与软著，形成”算法有论文支撑、系统有软著固化、方案有专利保护”的成果体系，具备落地转化潜力与社区治理推广价值。
